Getting Started

Install Vagrant Hostsmanager

The vagrant-hostsupdater plugin will modify your hosts file so that provisioned domains such as http://local.wp-puphpet.io works. To install it, run;

vagrant plugin install vagrant-hostmanager

Clone wp-puhpet

Open a terminal or your command line, create a new directory and cd into the directory.
Use this command to clone our repository: 

git clone git@github.com:web-dev-media/wp-puphpet.git

Install WordPress with composer

cd wp-puphpet/ && composer install

Start our wp-puphpet environment

vagrant up

Load WordPress on wp-puphpet in your browser

Visit http://local.wp-puphpet.io or http://172.23.0.55.
